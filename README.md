# Ansible Infrastructure Automation

## 📘 Overview
This repository contains Ansible playbooks and configuration files used to automate Ubuntu server patching, collect system resource data, and manage infrastructure across multiple environments.

Built with an Infrastructure-as-Code mindset, emphasizing idempotency, repeatability, and operational efficiency.

---

## 🛠️ Tech Stack
- 🔴 Ansible
- 🐧 Linux (Ubuntu)
- 🔐 SSH
- 📜 YAML
- ☁️ Infrastructure as Code (IaC)

---

## ✨ Key Features
- 🔄 Automated Ubuntu server patch management
- 📊 System resource utilization collection
- 🗂️ Inventory-based environment management
- ♻️ Idempotent playbook execution
- 🧩 Modular and reusable automation design

---

## Design Principles
- Idempotency-first automation
- Clear inventory segmentation
- Minimal manual intervention
- Secure SSH-based execution
- Reproducible infrastructure operations
